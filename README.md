#My Portfolio Advanced#

Welcome to my Advanced Portfolio, a modern and fully responsive web application showcasing my skills, projects, and experience as a Web Developer. This portfolio demonstrates my proficiency in front-end and full-stack development using modern web technologies.

🌟 Features

Responsive Design – Works seamlessly on desktop, tablet, and mobile devices.

Interactive Home Page – Includes animated text, buttons, and social icons.

About Me Section – Highlights my education, skills, and experience with engaging animations.

Skills Section – Dynamic progress bars for front-end and back-end technologies.

Projects Section – Grid layout displaying real-world projects with hover effects.

Contact Form – Fully functional form with modern UI, shadows, and animations.

Animations & Transitions – Smooth scroll, hover effects, and animated text transitions.

🛠 Technologies Used

Frontend: HTML5, CSS3, JavaScript, React.js

Backend: Node.js, Express.js

Database: MongoDB (for backend projects)

APIs: Integrated REST APIs for project data

Version Control: Git & GitHub

Additional Tools: CSS animations, responsive grid layouts, box shadows, and transition effects

🚀 How to Run Locally

Clone the repository:

git clone https://github.com/linganaboinasushmitha/My_Portfolio_Adavance.git

Navigate to the project folder:

cd My_Portfolio_Adavance

Open index.html in your favorite browser.

For full-stack functionality (Node.js + Express + MongoDB), make sure to install dependencies:

npm install

Then start the server:

node server.js
